# 电商用户增长分析项目

## 项目简介
通过分析电商平台用户的交易和行为数据，了解用户增长、留存、转化、复购等关键指标，支持运营优化。

## 使用工具
- SQL
- Python (Pandas, Seaborn)
- Tableau
- R (统计分析)

## 分析方向
- 用户留存曲线
- 用户分群（RFM 分析）
- 转化漏斗路径
- 用户生命周期价值（LTV）

## 文件结构
- `data/`：数据集
- `notebooks/`：分析代码
- `visualizations/`：可视化图表
- `reports/`：分析总结
